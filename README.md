### 技术栈 ###
* 开发语言:javascript
* 框架:vue
* 组件库: Ant-design
* 后端: [nodejs](https://github.com/WhiteSky12580/node)

### 启动 ###
**npm i**

**npm i node-sass@4.14.1 --save-dev**

**npm i sass-loader@10.1.0 --save-dev**

**npm run serve**
